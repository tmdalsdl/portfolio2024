Architecture

프로젝트 개요

AWS 클라우드 서비스를 활용한 개인 포트폴리오 웹사이트를 구축했습니다. 정적 웹 호스팅과 서버리스 아키텍처를 결합하여 확장 가능하고 비용 효율적인 웹 애플리케이션을 구현했습니다.

기술 스택

클라우드 및 인프라

  • AWS S3, CloudFront, API Gateway
  • AWS Lambda, DynamoDB
  • GitHub Actions

프론트엔드드

  • HTML/CSS
  • JavaScript

주요 구현 내용

CI/CD 파이프라인 구축

  • GitHub Actions를 통한 자동화된 배포 시스템 구현
  • S3 버킷으로 정적 웹사이트 자동 배포
  • CloudFront를 통한 콘텐츠 전송 최적화

서버리스 아키텍처 설계

  • API Gateway를 통한 RESTful API 엔드포인트 구축
  • Lambda 함수를 활용한 좋아요 기능 구현
  • DynamoDB를 활용한 좋아요 카운트 저장 및 관리

사용자 인터랙션 기능

  • 실시간 좋아요 카운트 업데이트
  • API Gateway GET/POST 요청 처리
  • 동적 UI 업데이트 구현

차별화 포인트

비용 효율적 아키텍처

  • 서버리스 아키텍처 채택으로 운영 비용 최소화
  • CloudFront를 통한 글로벌 콘텐츠 전송 최적화

확장 가능한 설계

  • API Gateway와 Lambda를 통한 서버리스 백엔드 구현
  • DynamoDB를 활용한 확장 가능한 데이터 저장소
