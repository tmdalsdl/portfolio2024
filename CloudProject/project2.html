서버리스 아키텍처 기반 포트폴리오

 Linux  Html  JavaScript  Github Actions  AWS

Architecture

프로젝트 개요

AWS 프리티어 서비스를 활용하여 개인 포트폴리오 웹사이트를 구축했습니다. GitHub Actions를 통한 자동화된 배포 시스템을 구축하여 코드 변경 사항이 실시간으로 반영되는 효율적인 포트폴리오 관리 환경을 구현했습니다.

기술 스택

클라우드 및 인프라

  • AWS S3, CloudFront, API Gateway
  • AWS Lambda, DynamoDB
  • GitHub Actions

프론트엔드

  • HTML/CSS
  • JavaScript

주요 구현 내용

CI/CD 파이프라인 구축

  • GitHub Actions를 통한 자동화된 배포 시스템 구현
  • S3 버킷으로 정적 웹사이트 자동 배포
  • CloudFront를 통한 콘텐츠 전송 최적화

서버리스 아키텍처 설계

  • API Gateway를 통한 RESTful API 엔드포인트 구축
  • Lambda 함수를 활용한 좋아요 기능 구현
  • DynamoDB를 활용한 좋아요 카운트 저장 및 관리

사용자 인터랙션 기능

  • 좋아요 버튼(이모지) 클릭 시 API Gateway POST 요청 처리
  • DynamoDB의 좋아요 수 실시간 업데이트
  • 페이지 로딩 시 API Gateway GET 요청으로 현재 좋아요 수 조회

차별화 포인트

효율적인 배포 환경

  • GitHub Actions를 활용한 자동화된 배포로 수동 작업 최소화
  • main brunch에 push만으로 실시간 포트폴리오 업데이트 가능
  • 변경 사항의 즉각적인 반영으로 포트폴리오 관리 효율성 향상

포트폴리오 접근성 향상

  • PDF 대신 웹페이지 형식을 채택하여 즉각적인 업데이트와 접근 용이
  • 모바일 환경에서도 원활한 확인 가능
  • 프로젝트별 좋아요 기능으로 관심도 파악 가능
