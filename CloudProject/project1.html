AWS 기반 확장 가능한 동영상 플랫폼 관리 시스템

 Linux  JavaScript  Django  Terraform  AWS

Architecture

프로젝트 개요

동영상 플랫폼의 관리자 페이지 개발을 담당하여 컨텐츠 업로드 및 관리 시스템을 구축했습니다. AWS 클라우드 서비스를 활용하여 확장 가능하고 안정적인 인프라를 구성했으며, CI/CD 파이프라인을 통해 효율적인 배포 환경을 구현했습니다

기술 스택

인프라 및 클라우드

  • AWS EC2, S3, RDS, IAM
  • Docker
  • Nginx

백엔드

  • Django
  • MariaDB

DevOps

  • Docker Hub
  • GitHub Actions

주요 구현 내용

클라우드 아키텍처 설계

  • EC2 인스턴스에 Nginx와 Django를 Docker 컨테이너로 구성하여 2-tier 아키텍처 구현
  • S3 버킷을 활용한 동영상 및 이미지 파일 저장소 구축
  • RDS를 통한 데이터베이스 관리 시스템 구축

CI/CD 파이프라인 구축

  • GitHub Actions를 활용한 자동화된 배포 시스템 구현
  • Docker Hub를 통한 컨테이너 이미지 관리
  • 환경 변수 자동 구성 및 보안 관리

자동화된 인프라 구성

  • EC2 인스턴스 자동 설정 및 필요 패키지 설치
  • RDS 데이터베이스 자동 구성 및 권한 설정
  • Django 마이그레이션 자동화

차별화 포인트

효율적인 인프라 선택

  • 제한된 사용자와 낮은 트래픽을 고려하여 Docker Container 기반의 2-tier 아키텍처 채택
  • 비용 효율적이고 관리가 용이한 구조 설계

자동화된 CI/CD 파이프라인

  • GitHub Actions를 활용한 완전 자동화된 배포 시스템
  • 단일 워크플로우로 인프라 구성부터 배포까지 자동화

보안성 강화

  • IAM Role을 통한 접근 권한 관리
  • GitHub Secrets를 활용한 민감 정보 보호
  • SSH 키 기반의 안전한 서버 접근
