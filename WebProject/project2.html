구현 이미지

프로젝트 개요

이 프로젝트는 2023년 10월 18일부터 10월 31일까지 진행된 팀 협업 기반의 쇼핑몰 웹 애플리케이션 개발 프로젝트로, Tomcat 서버를 활용하여 인터넷 화면을 구현하고, Controller와 Command 패턴을 사용해 Java 코드 구조를 정리하는 것을 목표로 하였습니다. 주요 기능으로는 회원 관리(회원가입, 정보 수정/삭제/조회), 상품 관리(등록, 수정, 검색, 리뷰), 주문 관리(장바구니, 결제 내역 조회) 및 고객센터(Q&A, FAQ)를 포함하며, 기본적인 쇼핑몰 기능을 담고 있으나 실제 결제 기능은 구현되지 않았습니다. 총 4인으로 구성된 팀에서 상품 관리 페이지(회원/비회원의 상품 상세보기 페이지, 관리자의 등록 및 수정 페이지), 메인 페이지의 상품 리스트 출력 및 회원 주문 목록 확인 등의 역할을 수행했습니다.

기술 스택

개발 환경

  • 개발 언어: Java, SQL, HTML, CSS, JavaScript
  • 개발 도구: Eclipse, SQL Developer, Brackets

데이터베이스

  • Oracle DB server

라이브러리

  • ojdbc6
  • jstl-1.2
  • mybatis-3.5.13
  • JQuery

주요 구현 내용

회원 관리 기능

  • 회원가입, 로그인, 정보 수정 및 삭제 기능 구현
  • 회원 데이터는 Oracle Database에 저장 및 관리
  • 사용자 인증 및 권한 관리를 통해 보안 강화

상품 관리 기능

  • 상품 등록, 수정, 삭제 및 검색 기능 제공
  • 상품 리뷰 작성 및 조회 기능 구현
  • 상품 데이터는 동적으로 출력되며 데이터베이스와 연동

주문 관리 기능

  • 장바구니 추가 및 주문 내역 조회 기능 개발
  • 주문 데이터는 Oracle DB에 저장 및 관리
  • 사용자별 주문 이력을 확인할 수 있는 페이지 제공

개선 방향

사용자 경험 향상

  • 현재 기본적인 UI에서 벗어나 직관적이고 시각적인 인터페이스 제공
  • 반응형 웹 디자인을 도입하여 다양한 디바이스에서 최적화된 사용자 경험 제공

기능 및 성능 개선

  • 결제 API(예: PayPal, Stripe) 연동을 통해 실제 결제 기능 추가
  • 데이터베이스 최적화를 통해 대규모 트래픽 처리 성능 개선

데이터 활용 및 분석

  • 사용자 구매 패턴 분석을 통해 추천 상품 시스템 도입
  • 데이터 시각화를 활용하여 판매 통계 및 트렌드 분석 가능하도록 개선
