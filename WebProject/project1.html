자바 퀴즈 천국

 Java  OracleDB  SQL

구현 이미지

프로젝트 개요

이 프로젝트는 3인 팀 협업으로 2023년 8월 24일부터 30일까지 7일간 진행되었으며, Java와 SQL을 활용해 콘솔 환경에서 텍스트 기반 퀴즈 게임을 개발하는 것을 목표로 하였습니다. 주요 기능으로는 회원 관리(추가, 수정, 삭제, 조회, 검색 및 순위 생성)와 퀴즈 문제 랜덤 출제, 정답 판별, 문제 등록/수정/삭제 및 오류 제보 등이 포함되며, 이러한 데이터를 효과적으로 관리하기 위해 Oracle Database를 활용하였습니다. 저는 이 프로젝트에서 퀴즈 문제 및 보기 랜덤 출제 기능을 구현하여 단순 암기로 높은 점수를 얻는 것을 방지하고 게임의 난이도를 조정하였으며, 문제 등록/수정/삭제 기능을 통해 관리자가 손쉽게 문제를 관리할 수 있도록 개발하였습니다.

기술 스택

개발 환경

  • 개발언어: Java, SQL
  • 개발 툴: Eclipse, SQL Developer

데이터베이스 및 라이브러리

  • DB: Oracle DB server
  • 라이브러리: ojdbc6

협업 도구

  • 협업 툴: Confluence, Jira

주요 구현 내용

회원 관리 기능

  • 회원 정보 추가, 수정, 삭제, 조회 및 검색 기능
  • 점수를 기반으로 순위 생성 및 조회 기능
  • 모든 데이터는 Oracle Database에 저장 및 관리

퀴즈 게임 기능

  • 퀴즈 문제 및 보기 랜덤 출제 기능
  • 정답 입력 및 판별 기능

문제 관리 기능

  • 관리자 로그인 시 문제 등록, 수정, 삭제 기능 제공
  • 오류 제보 시스템 구현

개선 방향

사용자 경험 향상

  • 현재 콘솔 기반 환경을 GUI로 확장하여 직관적이고 시각적인 인터페이스 제공
  • 퀴즈 카테고리 추가 및 난이도 선택 기능 구현으로 사용자 맞춤형 게임 환경 조성

기능 및 성능 개선

  • 각 문제에 키워드를 설정하여 약점 키워드와 관련된 문제를 추천하는 방식으로 구현
  • 코드 리팩토링을 통해 성능 최적화 및 유지보수 용이성 확보

데이터 활용 및 분석

  • 사용자 점수와 학습 패턴을 분석하여 개인화된 피드백 제공 (강점, 약점 분석하여 학습 방향성 제시)
  • 데이터 시각화를 통해 사용자의 학습 진행 상황을 한눈에 파악 가능하도록 개선 (그래프, 차트 등을 통해 점수 변화 추세, 목표달성률 등을 시각적으로 제공)
