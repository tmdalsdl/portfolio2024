프로젝트 개요

이 프로젝트는 유튜브 플랫폼을 벤치마킹한 동영상 플랫폼 개발을 목표로, 2023년 11월 14일부터 11월 30일까지 진행된 7인의 협업 기반 팀 프로젝트입니다. Spring, Ajax, Annotation을 활용하여 학습 내용을 복습하고 실무에 적용하는 것을 목표로 했으며, 회원 관리(등록, 수정, 삭제), 게시물 관리(영상 등록, 검색, 수정, 삭제, 댓글 및 좋아요/구독 기능), 고객센터(Q&A 및 문의)와 같은 주요 기능을 포함합니다. 비회원, 회원, 관리자 각각의 역할에 따른 기능 범위를 설정 및 구독 관리 등 세부적인 기능 구현을 통해 동영상 플랫폼의 기본적인 구조를 완성했습니다.
이 프로젝트에서 저는 상단 네비게이션 바(검색 기능 포함)와 좌측 사이드바를 개발하였으며, 모달(modal)을 활용한 로그인 화면, 회원 등록 기능을 구현했습니다. 또한, 관리자(admin)의 회원 관리 기능(정지, 탈퇴), 구독 관리 기능(등록, 취소, 리스트), 그리고 회원이 '좋아요' 버튼을 누른 영상들만 모아볼 수 있는 기능을 담당하여 사용자 경험을 향상시키는 데 기여했습니다.

기술 스택

개발 환경

  • 개발 언어: Java, SQL, HTML, CSS, JavaScript
  • 개발 도구: Spring(STS3), SQL Developer, Brackets

데이터베이스

  • Oracle DB server

라이브러리

  • ojdbc8
  • jstl-1.2
  • mybatis-3.5.13
  • JQuery
  • Jackson
  • Spring boot
  • Bootstrap

주요 구현 내용

회원 관리 기능

  • 회원가입, 로그인, 정보 수정 및 삭제 기능을 구현
  • 사용자 인증 및 권한 관리를 통해 보안성을 강화
  • 모든 데이터는 Oracle Database에 저장 및 관리

구독 및 좋아요 관리 기능

  • 구독 등록, 취소 및 리스트 조회 기능을 구현하여 사용자별 구독 상태를 실시간으로 관리
  • '좋아요' 버튼을 누른 영상들을 모아보는 기능을 통해 사용자 맞춤형 콘텐츠 제공

UI/UX 개발

  • 상단 네비게이션 바(검색 기능 포함)와 좌측 사이드바를 설계 및 개발하여 직관적인 인터페이스 제공
  • 모달 창을 활용한 로그인 및 회원가입 화면으로 사용자 편의성을 향상

개선 방향

사용자 경험 향상

  • 반응형 웹 디자인을 적용하여 다양한 디바이스에서 최적화된 UI 제공
  • 현재 기본적인 UI에서 벗어나 시각적으로 더 직관적이고 매력적인 인터페이스로 개선

기능 확장 및 성능 최적화

  • 동영상 스트리밍 성능을 최적화하여 대규모 트래픽 처리 능력을 강화
  • 추천 알고리즘을 도입해 사용자의 시청 패턴에 맞는 개인화된 콘텐츠 제공

데이터 활용 및 분석

  • 사용자 활동 데이터를 분석해 맞춤형 추천 시스템 도입 및 플랫폼 활용도 증대
  • 데이터 시각화를 통해 관리자에게 통계와 트렌드를 제공하여 운영 효율성 개선
